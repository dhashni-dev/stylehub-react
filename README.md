# StyleHub – Fashion E-Commerce Website

StyleHub is a modern and responsive fashion e-commerce web application built with React.js. The project is designed to provide a seamless shopping experience with product browsing, category filtering, product details, and cart management features.

## Features

* Modern and responsive user interface
* Browse fashion products by category
* Product details page
* Add products to cart
* Remove products from cart
* Update product quantity
* Search and filter products
* Mobile-friendly design
* Fast performance with React and Vite

## Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* React Router DOM
* CSS3 / Tailwind CSS
* Git & GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/dhashni-dev/stylehub-react.git
```

2. Navigate to the project directory

```bash
cd stylehub-react
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
